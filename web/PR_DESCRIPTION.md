# PR: 前端核心功能升级与网站理念构建

## 概述
本 PR 旨在提升 AgentMesh 的开发者体验与品牌深度，主要包括三个方面的改进：**Trust 可观测性增强**、**交互式调试工具**以及**核心理念页面的构建**。

## 核心变更

### 1. 核心理念与愿景 (Philosophy)
- **新页面**: 创建 `/philosophy` (核心理念)，系统阐述 AgentMesh 的定位——**数字员工管理系统、全网黄页与健康体检中心**。
- **内容架构**:
  - **三大支柱**: 神经系统 (协议)、记忆体 (存储)、生命体征 (健康监控)。
  - **激励机制**: 详细解读内在价值、经济激励、声誉系统与低门槛准入。
  - **视觉设计**: 采用高质感的 Glassmorphism 风格与动效，增强说服力。

### 2. 信任体系可视化 (Trust Observability)
- **多维评分**: 在 Agent 详情页展示 **可用性 (Availability)**、**可靠性 (Reliability)**、**性能 (Performance)** 三维雷达。
- **历史趋势**: 集成 `Recharts` 图表，直观展示信任分数的时间变化曲线。
- **事件日志**: 记录每一次心跳、调用成功/失败对分数的具体影响。

### 3. 交互式调试沙箱 (Invocation Playground)
- **即时调用**: 在详情页直接提供 JSON 编辑器与调用按钮，无需离开浏览器即可测试 Agent。
- **Relay 支持**: 针对内网穿透的 Relay Agent 增加专属标识 (紫色徽章) 与连接状态显示。

### 4. 基础设施优化
- **环境适配**: 引入 `NEXT_PUBLIC_API_URL`，支持本地与生产环境的灵活切换。
- **多语言支持**: 导航栏增加中文入口，优化部分文案。

## 测试指南
1. **启动服务**: 确保后端 (端口 8000) 与前端 (端口 3000) 均已启动。
2. **查看理念**: 点击导航栏的“核心理念”，检查文案排版与动效。
3. **测试图表**: 注册一个测试 Agent，进行几次调用（成功/失败），查看详情页的曲线变化。
4. **调试工具**: 使用 Playground 发送 JSON 请求，验证响应是否正确。
