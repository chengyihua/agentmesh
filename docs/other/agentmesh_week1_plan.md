## 🔧 技术栈选择

### 核心开发
- **语言**: Python 3.8+
- **Web框架**: FastAPI (轻量级、高性能)
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **测试**: pytest + unittest
- **文档**: MkDocs + Material主题

### 辅助工具
- **版本控制**: Git + GitHub
- **CI/CD**: GitHub Actions
- **代码质量**: black + flake8 + mypy
- **包管理**: Poetry
- **容器化**: Docker

### 部署环境
- **开发**: 本地环境
- **测试**: GitHub Actions
- **演示**: GitHub Pages + Vercel
- **生产**: AWS/Azure/GCP (未来)
